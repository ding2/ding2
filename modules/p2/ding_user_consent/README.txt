Description
-----------
ding_user_consent contains the privacy settings a user accepts or denies.

Version 1.0 contains only one setting but the module is prepared for multiple settings.

Supported settings
------------------
v1.0 
  loan_history_store Allow storage of loan history and personal lists of materials
v1.x
  

For now the settings are stored via Alma as certain loan categories.
This will change in the future.

Installation
------------
Install module along with compatible version of Alma
